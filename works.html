Hakase VS Kaiju (2023 Sep)

Submission to unity1week Game Jam お題「1ボタン」 .

A hyper-casual game where you play as a tiny inventor "Hakase-chan", who has to keep coming up new weapons to keep the Kaiju from taking over her toy island.
I worked with creators from the English-speaking Touhou fan community for this game. I was the designer and programmer.
The game placed 8th overall, and managed to get 1st in the Fun category, out of nearly 400 entries. (Results at the time of ranking announcements)

Amebrella (2023 Jun)

Submission to unity1week Game Jam お題「ふる」 .

An action platformer where you play as a robot girl with an umbrella, fighting against a gigantic titan that controls the weather. It featured textless tutorials and a very cinematic boss fight.
This is a collaboration between me and Japanese creators (a 2D artist and a composer) whom I met through the unity1week TeamUp!! program. I was the programmer here.
It placed 14th on average out of nearly 400 entries, and was a very fun experience for me overall.

When air hockey meets shooters. Given the short timeframe (4 days), it was quite a complete project. Notably, it got 1st in Visuals and Music, and 3rd in Story and Concept. I was the writer, designer, and programmer.

Sorting Battle (2022 Fall)

Final project for an undergraduate-level Machine Learning course.

This project is special in that it uses Unity's low-level Barracuda API to run an ONNX model trained using an external Python environment. The Python environment is built in a way that meets the same specs as the Unity application. To this end, unit testing is used extensively.
